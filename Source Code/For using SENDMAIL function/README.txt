
Khi su dung tinh nang sendmail de dat hang can phai configure lai mot so file lien quan nhu sau:

1. Copy thu muc sendmail vao duong dan C:\wamp64

2. Tuy theo phien ban WAMP da cai dat, copy file php.ini trong thu muc 64bit hoac 32bit 
   ghi de vao duong dan C:\wamp64\bin\apache\apache2.4.23\bin 
   
- Luu y: Neu dang su dung wamp phien ban 32bit thi duong dan thay vi wamp64 se la wamp

3. Khoi dong WAMP Server, vao PHP->PHP extension, bat php_opensll va php_sockets

4. Khoi dong lai WAP server, viec setup hoan tat.

- Luu y: phan sendmail nay hoat dong tren php version 5.6.25 hoac tuong duong, khong hoat dong
tren version 7. tro len
